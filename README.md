# Mind Dump（V1.1 双流版）

「零摩擦」的语音优先个人助理：说出即录入，自动区分「待办」与「点子」，分流存储。

- **待办**：有时间属性、需执行的任务（今日待办 / 日历）
- **点子**：灵感、笔记或随想（灵感库）

技术规格见 [notes/requirements.md](notes/requirements.md)。

---

## 启动步骤

### 1. 安装依赖

在项目根目录执行：

```bash
npm install
```

### 2. 本地开发

```bash
npm run dev
```

浏览器访问终端里给出的地址（通常是 `http://localhost:5173`）。  
**建议使用 Chrome 或 Edge**，以便使用浏览器原生语音识别（Web Speech API）。

### 3. 构建与预览（可选）

```bash
npm run build
npm run preview
```

`build` 会生成 `dist/`，`preview` 会本地预览打包后的页面。

## 常用操作

| 操作       | 说明 |
|------------|------|
| **录音**   | 长按中间圆形按钮开始录音，松开结束；转写后弹出「存为待办或点子」弹窗。 |
| **待办/点子** | 在弹窗里每条可点「待办」或「点子」；选待办可再选标签，最后点「保存」。 |
| **今日待办** | 底部第二个 Tab，只显示当天日期的待办。 |
| **灵感库** | 底部第三个 Tab，只显示点子。 |
| **日历**   | 按日期查看历史待办。 |
| **我的**   | 改昵称、管理任务标签。 |

数据全部存在浏览器 LocalStorage，无后端。
